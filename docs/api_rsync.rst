Treant syncronization
=====================
These are the API components of ``datreant.core`` for syncronizing
Treants locally and remotely.

.. _Sync_api:

Sync
----
The syncronization functionality is provided by the rsync wrapper function.
The function is used by the :py:meth:`datreant.core.Tree.sync` method.
